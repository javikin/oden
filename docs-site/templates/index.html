Stack Templates

Pre-configured MCP bundles for popular tech stacks. Copy the configuration, paste into your .mcp.json, and you're ready.

Next.js Full Stack

Complete setup for Next.js applications with Supabase, Figma design integration, Vercel deployment, and E2E testing.

ℹ️
This template includes 5 MCPs. Replace YOUR_PROJECT_REF and YOUR_ACCESS_TOKEN with your actual Supabase credentials.
json - .mcp.json 
{
  "mcpServers": {
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    },
    "figma": {
      "type": "http",
      "url": "https://mcp.figma.com/mcp"
    },
    "supabase": {
      "command": "npx",
      "args": [
        "-y",
        "@supabase/mcp-server-supabase@latest",
        "--read-only",
        "--project-ref=YOUR_PROJECT_REF"
      ],
      "env": {
        "SUPABASE_ACCESS_TOKEN": "YOUR_ACCESS_TOKEN"
      }
    },
    "vercel": {
      "type": "http",
      "url": "https://mcp.vercel.com/"
    },
    "playwright": {
      "command": "npx",
      "args": ["-y", "@anthropic/mcp-server-playwright@latest"]
    }
  }
}

Included MCPs: Context7 · Figma · Supabase · Vercel · Playwright
