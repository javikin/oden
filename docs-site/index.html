Quick Start

Get up and running in under 2 minutes.

Install Oden Forge

Clone the repository and run the installer.

bash 
git clone https://github.com/javikin/oden.git my-project
cd my-project
./install.sh

Install MCPs from Claude Code

Use the built-in command to install MCPs interactively.

Claude Code 
# Install essential MCPs
/oden:init-mcp essential

# Or install by category
/oden:init-mcp design
/oden:init-mcp testing

Or configure manually

Add MCP configurations directly to your .mcp.json file.

json - .mcp.json 
{
  "mcpServers": {
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    },
    "figma": {
      "type": "http",
      "url": "https://mcp.figma.com/mcp"
    }
  }
}

Restart Claude Code

MCPs load on startup. Restart Claude Code and verify with /mcp.
