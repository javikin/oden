MCPs / Context7

📚 Context7

Essential Type: HTTP Free API Key

One-Click Install

Run this command inside Claude Code to install Context7:

Claude Code 
/oden:init-mcp essential

Manual Configuration

Add this to your .mcp.json file in the project root:

json - .mcp.json 
{
  "mcpServers": {
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}
ℹ️
Context7 offers a free API key. Visit context7.com to get yours. If you use it without a key, it still works but with rate limits.

What It Does

  • Retrieves up-to-date documentation for any npm/pip/crate library
  • Provides real code examples from official docs
  • Works with Next.js, React, Vue, Supabase, Tailwind, and hundreds more
  • Keeps Claude Code from hallucinating outdated API patterns

Usage Examples

Once installed, Claude Code will automatically use Context7 when researching libraries. You can also prompt it directly:

Prompt examples 
# Ask about a library using Context7
"How do I use the new App Router in Next.js 15?"

# Get updated API patterns
"Show me how to create a Supabase RLS policy"

# Research a specific feature
"What's the correct way to use React Server Components?"

Best Practices

  • Install Context7 on every project - it prevents outdated code suggestions
  • Mention specific library versions when asking questions for better results
  • Use /oden:research docs [lib] [topic] for structured research

Included In Templates

Context7 is included in all stack templates:

  • Next.js Full Stack
  • React Native Mobile
  • Backend / API
  • Full Stack Enterprise
