Migration Steps

Step-by-step guide to migrate to v2.2.1

1. Update Package

Update to the latest version via NPM.

bash 
npm update -g oden-forge

# Verify installation
oden-forge --version  # 2.2.1

2. Update Your Workflow

Replace obsolete commands with new orchestrator.

diff 
- /oden:dev frontend
+ /oden:work

- /oden:dev debug
+ /oden:debug

- /oden:dev test
+ /oden:work

3. Try New Debug System

Experience the revolutionary queue-based debugging.

Claude Code 
# Start interactive session
/oden:debug

# Add tasks to queue
debug> add "investigate login issues"
debug> add "fix validation errors"
debug> status

4. Verify Migration

Ensure everything is working correctly.

Claude Code 
# Check available commands
/oden:help

# Should show 8 essential commands:
# Setup: /oden:init, /oden:mcp, /oden:help
# Pre-Dev: /oden:architect, /oden:prd, /oden:epic
# Development: /oden:work, /oden:debug, /oden:sync
